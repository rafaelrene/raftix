# Raftix

Raftix is a custom linux distribution based on [Artix Linux](https://artixlinux.org/).
It is meant to be used as my personal operating system and is meant to be my personal distribution for my use only.
When making decisions, my needs and wants are the only ones that matter.
There's no need for generalizations in case some people want to use some things differently.

The purpose of this distribution is to skip all the hassle of setting up a new system.
Basically, it should be a distro that starts pre-configured exactly to my liking and nothing else.

## Plans

All plans are in the `./.plans/` directory. Each plan is a directory with a name describing the main goal of the plan.
It must contain a `README.md` file with a description of the plan. Everything else depends on the plan itself.
If plans is complex, it can be broken down into multiple smaller files and the main README.md can serve as a summary and main entry point.
